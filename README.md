# AI Climate Tipping Point Detection System

An AI-powered system that detects potential climate tipping points globally using machine learning.

Features:
- Global climate risk scanning
- AI anomaly detection
- Interactive world map
- Climate variance visualization
- 30-day tipping point projection

Tech stack:
Python
Streamlit
Scikit-learn
Plotly
Climate datasets (NASA / NOAA)

Run locally:

pip install -r requirements.txt
streamlit run app.py
